##  Portfólio IBM - ETEC/P-TECH 2025 (2DS)

Este repositório foi criado com o objetivo principal de **registrar, documentar e comprovar** as **horas** e **atividades** exigidas pelo programa P-TECH da IBM e pela disciplina de Desenvolvimento de Sistemas (DS) do 2º ano da ETEC.

Serve como um arquivo digital completo para auditoria e verificação do cumprimento dos requisitos do projeto integrador e das horas complementares/curriculares estabelecidas pela parceria.
